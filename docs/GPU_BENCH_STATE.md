## Docker context gotcha (CRITICAL for next session)

**Docker Desktop hijacks the `docker` CLI.** Default context was `desktop-linux` (VM, no GPU). Switch to `default` (native dockerd) for GPU access:
```
docker context use default
```
Verify: `docker info | grep -i runtime` should show `nvidia` in Runtimes list.

CDI spec at `/etc/cdi/nvidia.yaml` and `/var/run/cdi/nvidia.yaml` - both generated by nvidia-ctk, both valid. Only native dockerd sees them.

## apparmor gotcha on Ubuntu 24.04

`kernel.apparmor_restrict_unprivileged_userns` must be 0 for CUDA to init from venv Python. User's setting - they flip it manually per session. NOT persisted (their choice for security). graphstore's preload detects this and raises a targeted error.
